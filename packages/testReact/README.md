# JsonUI Test React App

This package is a lightweight local app for validating JsonUI behavior in a plain React environment.

## Run

```bash
yarn workspace jsonui-test-react dev
```

## Build

```bash
yarn workspace jsonui-test-react build
```

## Type Check

```bash
yarn workspace jsonui-test-react type-check
```

## Scope

Use this app for quick manual checks of:

- JSON model rendering
- store/path simplification behavior
- inline validation behavior
- custom action/modifier integration

For full API reference and documentation, see `packages/docs-site` and https://jsonui.org.
