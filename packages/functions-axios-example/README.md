# JsonUI Functions Axios Example

This package demonstrates custom JsonUI actions/modifiers with Axios-based API calls.

Use this example when you want to:

- implement custom `$action` handlers for async requests
- compose request/response transformations with JSON models
- test data flow in Storybook

## Run

```bash
yarn workspace @jsonui/functions-example storybook
```

## Notes

- API calls in this package are example-oriented and may use mock/fake endpoints.
- Inspect browser network logs while interacting with the stories.
- JsonUI core features such as simplification, inline validation, and JSONata semantics are documented in the docs-site.
